##Memo board

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

This project uses:

- NodeJS
- ReactJS
- Redux
- Redux Sagas
- Webpack 2

##Usage

use node LTS `v6.10.3`

Run `yarn` or `npm i`

Run tests:

`yarn test` or `npm test`

run app:

`yarn start` or `npm start`

go to:

`http://localhost:3000`
